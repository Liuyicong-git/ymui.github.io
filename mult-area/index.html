MultArea 多数据按钮组切换

# 基础用法

组件效果
当前值： []
0/20 
<template>
  <div>
    当前值： {{value}}
    <mult-area ref="multArea" :labels="['运单号', '工单编码', '差错编码']" v-model="value" :default-index="0" />
    <el-button @click="reset">重置</el-button>
  </div>
</template>
<script>
export default {
  data() {
    return {
      value: [],
    };
  },
  methods: {
    reset() {
      this.$refs.multArea.reset();
    },
  },
};
</script>

参数 说明 类型 可选值 默认
labels 按钮组名称 Array []
defaultIndex 默认选中按钮index Number 0
blankValue 是否去除空格 Boolean true/false true
max 最大输入数据个数 Number 20
